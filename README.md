App主要功能有：
 1. 主页：添加和删除便签单条数据、清空便签所有数据、设置置顶、紧急、待办TODO 便签等；垃圾回收站页面：清空便签所有数据； 还原单条数据；添加便签页面：添加， 显示详情 撤销修改；
 2. 资料页：本地PDF、 EPUB、 TXT 文档或Ebook阅读；
 3. 音视频页：本地音乐和视频文件列表浏览、音视频播放；
 4. 服务页：搜索天气、搜索网页-网上冲浪、锁屏等；
 5. 设置页等。 打开App无广告。
